Privacy Policy

UPS Parking is an application designed to help users check parking availability and reserve parking spaces at Universidad Politécnica Salesiana. We respect user privacy and aim to minimize the collection of personal data.

Information We Collect

UPS Parking does not collect personal information such as name, email address, phone number, or location data from users.

The application may process limited operational information necessary for providing parking services, such as parking space availability and reservation status stored in the system database.

How We Use Information

The limited information processed by the application is used only to:

  • Display available parking spaces
  • Allow users to reserve parking spots
  • Manage parking occupancy within the system

Third-Party Services

UPS Parking may communicate with a backend API operated by the developer to retrieve parking availability data and manage reservations. This backend service does not use the information for advertising or marketing purposes.

Data Storage

Parking availability and reservation information may be stored in a secure database managed by the application infrastructure. This data is used strictly for operational purposes related to parking management.

Data Security

We implement appropriate technical and organizational security measures to protect system data and prevent unauthorized access.

User Rights

If you have questions regarding this privacy policy or the operation of the application, you may contact the developer using the contact information provided below.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any updates will be published on this page.
